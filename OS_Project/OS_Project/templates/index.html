Project Overview

CPU Scheduling Simulator Web Application

The CPU Scheduling Simulator is a web-based application developed to simulate and visualize different CPU scheduling algorithms used in Operating Systems. The project helps users understand how processes are managed and executed by the CPU through interactive calculations, graphical representations, and ready queue visualization.

The simulator supports the following algorithms:

  • FCFS (First Come First Serve)
  • SJF (Shortest Job First)
  • SRTF (Preemptive SJF)
  • Priority Scheduling
  • Round Robin Scheduling

Users can select an algorithm, enter process information such as arrival time, burst time, priority, and time quantum, and generate scheduling results instantly.

The application displays:

  • Gantt Chart visualization
  • Process execution table
  • Waiting Time and Turnaround Time
  • Average performance calculations
  • Ready Queue state at each execution step

The project is developed using:

  • Python Flask for backend scheduling logic
  • HTML, CSS, and JavaScript for frontend UI design

The main objective of this project is to provide an attractive, user-friendly, and educational platform for learning CPU scheduling algorithms through real-time simulation and visualization.

Why use it?

Because understanding CPU scheduling becomes far easier and more engaging with real-time process simulation, animated Gantt chart visualization, live ready queue tracking, and a modern interactive dashboard. Instead of learning scheduling algorithms only through theory and calculations, this simulator allows users to visually observe how processes are executed, queued, preempted, and completed step by step in an intuitive and user-friendly environment.
